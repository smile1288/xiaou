## 5.1 余额充值（平台内购买U）

从小U平台内购买U，方便快捷，安全高效，无手续费

### 📝 操作流程

1. **入口**：平台底部 → 交易所
2. **选择商家** → 立即买入
3. **输入购买数量** → 付款方式 → 确认下单

> ##### 🚨 重要提醒
>
> **一定等待商家接单后方可转账！！！**
{: .block-danger }

4. 商家接单后进入付款时间
5. 付款后**上传付款凭证**
6. 待商家确认后放币到你余额内

> ##### 💡 小贴士
>
> - 付款前请先通过在线联系确认商家是否在线
> - 除付款外，其他所有聊天内容请不要离开小U折扣平台！
> - 期间出现任何问题及时联系客服介入
{: .block-tip }

### 📝 操作步骤（以欧易为例）

#### 第一步：在欧易购买U

1. 在欧易APP内购买USDT

#### 第二步：获取充值地址

1. 打开小U平台
2. 点击 **我的 → 充值提现**
3. 输入充值数量 → 确认
4. **保存二维码**（或复制二维码下方链接）

#### 第三步：提币到平台

1. 打开欧易APP
2. 点击 **提币 → 提取数字货币**
3. 币种选择 **USDT**
4. 收款人选择 **其他交易所或钱包**
5. 选择网络 **Tron (TRC20)**
6. 地址输入平台复制的地址
7. 输入提币金额
8. 确认提币

> ##### ⚠️ 手续费说明
>
> 注意右下角的网络费用，实际到账金额 = 提币金额 - 网络费用
> 网络费用是欧易收取的与平台无关，买币时记得把手续费也算上
> 欧易提一次约 **2.3U**
{: .block-warning }

#### 第四步：上传凭证

1. 提币成功后保存截图
2. 回到小U平台订单页面
3. **上传截图 → 立即提交**
4. 等待 **3-5分钟** 即可到账

## 5.3 平台余额提现

### 📝 操作流程

1. **入口**：我的 → 充值提现
2. 输入提现数量
3. 确认钱包地址 → 确认
4. **发起提现后联系客服审核**

### 📋 提现规则

| 项目 | 说明 |
|:-----|:-----|
| 最低提现 | **50U** 起 |
| 提现手续费 | **2U** |
| 到账时间 | 显示提现成功后 **2小时内** |

> ##### ⚠️ 问题处理
>
> 若遇到提现失败、提现成功未到账情况，请及时联系客服处理！
{: .block-warning }
